# Jupyter to Typst converter

## Features

- automatic image download/extract
- nice template
- support for KaTeX in markdown convert (not complete)

Look for demos in [test](test) folder.

## Usage

```
jupyter2typst <input> [-o <output>] [-i <img-path>]
```
